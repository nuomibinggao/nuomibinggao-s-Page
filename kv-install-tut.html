How to Install KeyViewer Profiles

← Back to Downloads

1. Download a Profile

Go to the KeyViewer Downloads page and click the download button for the profile you want (e.g., 8K, 10K, 16K, etc.). The file will be saved as a .json file.

2. Open KeyViewer Config

Launch ADOFAI, make sure to install KeyViewer 4.0.0 or later. In game, press Ctrl + F10 (Default) to open Unity Mod Manager, look for the KeyViewer mod, and click the options button.

3. Import the Profile

  • In KeyViewer, click Import Profile.
  • Click it and select the downloaded .json file from your computer.
  • The profile should now appear in your list. Select it to activate.

4. Adjust and Save

When imported, the font will not load, you need to manually change the font in the mod config. Recommended fonts can be downloaded here. You may also need to adjust the position (reffered to Offset in mod config) to make the profile appear on your screen depending on your resolution, you can ignore this if you are on 2K (2560 × 1440) resolution. Make sure to save your changes by pressing Save at the bottom of the mod manager window.

Note: If you have trouble importing, make sure you are using KeyViewer version 4.0.0 or later. If the profile does not appear, check that the file was not renamed or corrupted during download.
